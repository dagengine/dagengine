# Code of Conduct

## Our Pledge

We pledge to make dagengine a harassment-free community for everyone, regardless of age, disability, ethnicity, gender identity, experience level, nationality, race, religion, or sexual orientation.

## Standards

**Positive Behavior:**
* Be respectful and empathetic
* Give and accept constructive feedback gracefully
* Take responsibility for mistakes
* Focus on community benefit
* Help others learn dagengine

**Unacceptable Behavior:**
* Harassment, trolling, or personal attacks
* Sexualized language or unwelcome advances
* Publishing others' private information
* Disruptive, spam, or bad faith conduct
* Any behavior inappropriate in a professional setting

## Scope

Applies to: GitHub repos, documentation, communication channels, events, and public representation of dagengine.

## Reporting

Report via:
1. **GitHub** - Use "Report abuse" feature or contact maintainers
2. **Private message** - DM a project maintainer on GitHub

**Privacy:** Reports handled confidentially. Reporter identity protected unless agreed otherwise.

## Enforcement

1. **Correction** - Private warning for minor violations
2. **Warning** - Formal warning with consequences for repeated behavior
3. **Temporary Ban** - No community interaction for specified period
4. **Permanent Ban** - Permanent removal from all community spaces

Maintainers review reports within 48 hours and decide action within 7 days.
